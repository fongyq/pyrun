# requirements

- jsdiff

  - https://www.npmjs.com/package/diff

  - `npm install diff`

- diff2html

  - https://www.npmjs.com/package/diff2html

  - `npm install diff2html`
  
- codemirror
  
  - https://codemirror.net/5/

note：直接引用 npm 安装之后的 node_modules 目录在线上会报 404，所以把需要的包单独拷贝到了 npm_modules 目录再使用。
